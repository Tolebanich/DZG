Молитва Богу-машине который благословил сети Edge-core

Для литургии Богу-машине нам понадобится консольный кабель RS232 для Edge-core аналогичный как для коммутаторов Allied Telesis и ТВ тюнеров (мама-мама).

Без данного кабеля Бог-машина не услышит наши молитвы

Вознесение молитвы для просмотра текущего состояния духа Бога-машины и сразу же начнём литургию через конфигурацию

  • show running-config
  • configure

Далее,молитвой добавим необходимые VLAN для упрощения работы духа-машины

  • VLAN database
  • vlan 207 name Management
  • vlan 226 name Clients
  • exit

Теперь необходима литургия настройки порта для обращения к Богу-машине и великой сети

  • interface ethernet 1/25
  • switchport allowed vlan add 1 untagged
  • switchport allowed vlan add 207,226 tagged
  • switchport mode trunk
  • exit

Для обращения к этому духу машины удалённо нужно настроить управляющий VLAN и указать адрес этой машины в великой сети

  • interface vlan 207
  • ip address 192.168.5.5 255.255.255.0
  • exit
  • ip default-gateway 192.168.5.1

Для работы непосвященных необходимо настроить им порты, дабы и до них доходило благословение

  • interface ethernet 1/1-24
  • switchport native vlan 226
  • switchport allowed vlan add 226 untagged
  • switchport broadcast packet-rate 1024
  • exit

Сохраним священные слова молитвы и проверим правильность обращения к духу-машине

  • show running-config
  • copy running-config startup-config
  • config.cfg

После проведённой литургии машина будет готова к работе. И непосвященные получать частицу благословения Бога-машины
